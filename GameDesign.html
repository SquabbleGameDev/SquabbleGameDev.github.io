Process

In the first demo, we actually design 3 level, Level 1 & 2 is in the bunker and level 3 is outside the bunker.

After testing it out, we felt the game still lacking something.After discussion, we found out that our game lack of these things.

1. Storyteling

Our game only have text dialogue for storytelling, it doesn't create enough interaction with the audience.

We decided to create animation for storytelling and added voice line into it.

2. Player Navigation

We found out our game instruction is not clear. Access to shortcut key is troublesome.

We decided to put the instruction and shortcutkey in the game scene, so that player know about them. The shortcut key can be toggle-on and off up to player's preference.
